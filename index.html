PROJECTS

PokemonPartyAPI

I wanted to do a slightly different take on the Pokemon API. Designed to Fetch information from the RESTful PokemonAPI, the search takes both names and numbers, and has a lot of built in functionality to handle user error and give feedback. After the user builds their party of 6 pokemon the oldest pokemon is "transferred" out, just like the original game.

CSS Snorlax

This Snorlax was built entirely from css shapes. I had fun implementing the animated Zzz effect using cascading positioning and animations.

Wizarding-World Page Recreation

This challenge involved using only HTML & CSS to do a recreation of a complex webpage. I utilized keyframe animations to recreate the loading effects from the original and got deep into embedding special fonts and complex positioning tools.

The Owlery Restaurant

I designed this website with only HTML and CSS before beginning my classes at Eleven Fifty. It needed to be implemented on the restaurant's legacy Tumblr setup, which presented some challenges. They needed the ordering and telephone buttons front and center, and a clear system for displaying various menus. This site is currently live as the primary website for the business.

Beast:Mode

I worked on a team of 4 developers to create this full stack website for Obstacle Course Racers to store their race data, view it on a calendar, and message with other users on a custom built message board. I used React, HTML 5, CSS 3, Bootstrap 4, Node.js, Express, Sequelize, PostgresSQL, and JSON Web Token. The site is deployed on Heroku, and features user authentication, protected views, and all CRUD functionality.

Table Tempo

With my history in restaurants, I wanted to make a lightweight app that would give restaurants a realtime seating waitlist, which offers constantly refreshing list of parties sorted by time and with visual cues if they are overdue. The parties' data can later be searched and visualized in a report. The restaurant has linked staff accounts they can manage with Admin permissions. The site was built with React, Material UI, Typescript, Node, Express, and Sequelize.
